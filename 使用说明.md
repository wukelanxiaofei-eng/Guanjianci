# OZON 关键词标签 & 产品简述生成工具 — 使用说明

本工具在自己电脑上运行：填好两张表 → 自动生成俄语 `#标签` 和俄语简述 →
**人工审核**后手动粘贴到 OZON 后台。v1 不连接 OZON API，安全简单。

---

## 一、第一次安装（每台电脑只需一次）

1. **装 Python**：到 https://www.python.org 下载 Python 3.9 或更高版本安装。
   （Windows 安装时记得勾选 “Add Python to PATH”。）
2. **装依赖**：把本文件夹整个放到电脑上，打开命令行（Windows 用 PowerShell / CMD），
   `cd` 进到本文件夹，运行：
   ```
   pip install -r requirements.txt
   ```

## 二、配置 API Key（任选一种）

工具用阿里云百炼（通义千问）生成内容，需要一个 API Key（`sk-` 开头）。
**Key 不要发到聊天群、不要写进截图。** 三种配置方式任选其一：

- **方式 A（推荐给团队统一配置）**：把 `config.example.json` 改名为 `config.json`，
  把里面的占位文字换成真实 Key。
- **方式 B**：设系统环境变量 `DASHSCOPE_API_KEY`。
- **方式 C**：每次在网页左侧栏直接粘贴 Key（只在本次使用，不保存）。

> Key 在哪拿：阿里云百炼控制台 → API-KEY → 创建 API Key。新用户每个模型约 100 万
> 免费 token、90 天有效，够跑很多产品。额度用完调用会报 `429`，届时充值即可。

## 三、运行

在本文件夹的命令行里运行：
```
streamlit run app.py
```
浏览器会自动打开工具页面（没自动开就手动访问命令行里给出的本地网址）。

---

## 四、日常使用步骤

1. 左侧栏点 **“下载 SKU 填写模板”**，按模板把每个产品一行填好
   （货号、产品名称/品类、码段、材质、人群、颜色、场景、季节、图片链接）。
   - 这些可以用中文填，工具会自动产出俄语结果。
2. 从 OZON 后台导出 **当期热搜词表**（搜索词报告，xlsx）。
3. 在页面上分别上传：① SKU 信息表　② 热搜词表。
4. 左侧选模型：**简述建议 `qwen-plus`**；要更好质量用 `qwen-max`；
   想更省可把简单任务换 `qwen-flash`。
5. 点 **“开始生成”**，等待跑完。
6. **逐条审核**每个产品的标签和简述（重点看“风险提示”），确认无误后：
   - 复制标签 → 粘贴到 OZON 卡片的 **Хештеги** 字段；
   - 复制简述 → 粘贴到 **Annotation** 字段。
7. 也可点 **“下载全部结果”** 拿到一张待审核 Excel，集中审核。

## 五、工具已自动遵守的 OZON 规则

- 标签：俄语、`#` 开头、词内用 `_`、仅字母数字下划线、单标签 ≤30 字符、最多 30 个；
  **自动去掉词序颠倒的近义重复**（如 `мужские_кроссовки` 与 `кроссовки_мужские` 只保留一个）。
- 简述：**5000–6000 字符**（太短自动扩写、太长在句末截断），全方位描述特点、码段、
  人群、场景、季节、颜色，并把相关高热搜词**自然穿插**，不堆砌。
- 选词：先按性别/品类/材质过滤掉不相关的词，再按热度排序——而不是闷头选最热的
  （比如男鞋不会去用最热的女款词）。

> ⚠️ 工具是辅助，**生成结果必须人工审核**：模型可能把尺寸、材质等细节写错。
> “风险提示”列会标出偏短、标签不足、疑似堆砌、密度偏高等情况，优先核对这些。

## 六、常见问题

- **报 401 / Key 无效**：检查 Key 是否填对、是否已在百炼控制台启用。
- **报 429**：免费额度用完了，去百炼充值开通按量付费。
- **读表失败**：确认上传的是 `.xlsx`；热搜词表用 OZON 原始导出即可（非标准格式工具会自动兼容）。
- **简述偶尔不足 5000 字符**：再生成一次，或改用 `qwen-max`。
